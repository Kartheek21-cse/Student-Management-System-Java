# Student Management System (Java + JDBC + PostgreSQL)

A console-based Java application to perform CRUD operations on student records stored in a PostgreSQL database using JDBC.

## Features
- Add, update, delete, fetch student records.
- Console menu interaction.

## How to Run
- Make sure PostgreSQL is running.
- Compile and run the project via Eclipse or command line.

## Database
Run `student.sql` in your PostgreSQL to create the table.
